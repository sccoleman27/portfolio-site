A little about me.

I live just outside of Boulder, Colorado and enjoy an active lifestyle. My journey into tech started with a degree in psychology. As I became fascinated with the psychology of design, I realized how important an asset it would be to learn to build my own designs using technical tools. So I began with learning how to code.

That curiosity then opened a door that could never be shut. Once I built my first website, I realized I found something that gave me the freedom & versatility I was looking for in my work. Now I've made a full switch to web development and design, where I can use my organization skills and eye for detail to write clean, elegant code.

Hand-Coded HTML

My focus is writing clean, well formatted, semantic HTML5 by hand to make sure that the content is easy to read, easy to collaborate, trouble shoot and accessible.

Well-Organized CSS

I pride myself on writing CSS that is easy to read and build on. I focus on keeping my CSS lean and fast to load, and I make it a habit to stay up to date on current best practices.

Ease Converting Designs into Code
